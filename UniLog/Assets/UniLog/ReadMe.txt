UniLog 1.1.0

■ クイックスタート

1. インポート後の設定

プロジェクトのアセンブリ定義から Assets/UniLog/UniLog.asmdef を参照する。

2. ログ出力のコードを書く

// using UniLog; が必要
Log.Warning("警告メッセージ");
Log.Info("何らかの情報");

３．ログ出力の設定（任意）

Unity エディタで Assets/UniLog/Resources/UniLogSettings を選択して、インスペクタで適宜設定を行う。
